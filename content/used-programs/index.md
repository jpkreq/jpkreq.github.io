# 지진관측에 도움되는 프로그램들

Tip. 설치가 안 될 경우 `제어판` → `시계 및 국가` → `국가 또는 지역` → `관리자 옵션` → `시스템 로캘 변경` → `일본어(일본)`으로 번경

주의. 세계 언어 지원을 위해 Unicode UTF-8 사용 체크해제
